# Screen — Evening Theme (THÈME / SOIRÉE)

Evening Theme sets the context for the current night.

## Core Features

- Drink type selector.
- Mood/role selector.
- Average drink price.
- Calendar planning/logging.
- Current theme display on Map and Stats.

## Drink Type Selector

Prompt:

```text
Je bois que...
```

Options can include:
- Beer
- Cocktails
- Shots
- Wine
- Mixed
- Nothing yet

The selected drink type affects:
- Current session context.
- Stats grouping.
- Drunk score context.
- Average price assumptions.

## Mood / Role Selector

Prompt:

```text
Je suis que...
```

Options can include:
- Chill one
- Dancer
- Instigator
- Babysitter
- Mystery
- Going home soon

The selected role affects:
- Current night profile.
- Map/session labels.
- Historical night records.

## Average Drink Price

The user can set or edit the average drink price for the night.

Used for:
- Spending estimates.
- Stats.
- Venue/theme context.

## Calendar Integration

Calendar support can be used to:
- Plan future nights.
- Log past nights.
- Attach a theme to a planned event.

## Data Usage

Theme data feeds into:
- Map labels.
- Stats grouping.
- Drunk score context.
- Session history.

The user can update the theme during the night.
